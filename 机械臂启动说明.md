# 机械臂启动命令说明

> 本目录 `smq&jgy` 的真机操作入口。流程来自桌面 `deoxys.txt`，已封装为 `scripts/` 下可执行脚本。  
> **不修改** `wty/deoxys_control` 等外部目录中的任何文件。

## 路径注意

目录名含 `&`，所有 `cd` 命令请加引号：

```bash
cd "/home/host5010/workspaces/smq&jgy"
```

## 前置条件

| 项目 | 要求 |
|------|------|
| conda 环境 | `franka_mani` |
| deoxys 路径 | `/home/host5010/workspaces/wty/deoxys_control/deoxys` |
| 机器人配置 | `config/charmander.yml` |
| CPU 模式 | **performance**（deoxys 要求，不可用 powersave） |
| 夹爪 | **Franka 原生 Hand**（Terminal 2 的 `auto_gripper.sh`） |
| 遥操设备 | SpaceMouse（本机检测到 product_id **50746**） |

## 方式 A：分终端启动（与桌面流程一致，推荐调试）

先进入项目目录：

```bash
cd "/home/host5010/workspaces/smq&jgy"
```

### Terminal 1 — 机械臂（长线服务，勿关）

```bash
bash scripts/start_arm.sh
```

等价于桌面：

```bash
cd ~/workspaces/wty/deoxys_control/deoxys
conda activate franka_mani
./auto_scripts/auto_arm.sh config/charmander.yml
```

### Terminal 2 — 夹爪（长线服务，勿关）

```bash
bash scripts/start_gripper.sh
```

启动成功应看到 **`Gripper homing complete`**。若出现 `Connection timeout`，夹爪无法控制，需检查机器人 FCI 与网络。

### Terminal 3 — 遥操测试（短线命令）

```bash
bash scripts/teleop_test.sh
```

| SpaceMouse 操作 | 效果 |
|-----------------|------|
| 移动/扭转 | 平移/旋转机械臂 |
| **左键长按** | 闭合夹爪 |
| 松开左键 | 张开夹爪（默认每帧发 open） |
| 右键 / ESC | 退出（可能触发 Python 报错，见下方说明） |

若末端**自己旋转**，改用仅平移模式：

```bash
bash scripts/teleop_test.sh OSC_POSITION
```

### Terminal 4 — 低维数据采集（无相机）

```bash
bash scripts/collect_lowdim.sh
```

### 机械臂 + RealSense 同步采集

```bash
bash scripts/collect_sync_realsense.sh
```

### 相机测试

```bash
bash scripts/detect_cameras.sh    # 列出 serial，生成测试命令
bash scripts/free_cameras.sh      # 若报 Device busy，先释放相机
bash scripts/test_camera.sh       # SSH 无显示器时自动无头测试，PNG 存 data/camera_test/
```

**不要用** `deoxys/examples/test_camera.py` 直接跑 SSH 终端（会报 `could not connect to display`）。  
若之前崩溃过，必须先 `bash scripts/free_cameras.sh` 再测。

### 关节复位

```bash
bash scripts/reset_robot.sh
```

## 方式 B：一键后台启动

适合已确认硬件正常、想少开终端时：

```bash
cd "/home/host5010/workspaces/smq&jgy"
bash scripts/start_robot.sh
```

会在后台同时拉起机械臂 + 夹爪，日志：

- 机械臂：`logs/deoxys_arm.log`
- 夹爪：`logs/deoxys_gripper.log`

然后新开终端：

```bash
bash scripts/teleop_test.sh
# 或
bash scripts/collect_sync_realsense.sh
```

停止后台服务：

```bash
bash scripts/stop_robot.sh
```

## 常见问题

### 1. 遥操时报 `IndexError: array is 0-dimensional`

按了 SpaceMouse **右键**或 **ESC** 后，`input2action` 返回 `action=None`，`run_deoxys_with_space_mouse.py` 未做保护。  
**处理**：遥操时不要按右键/ESC；要退出直接在终端 `Ctrl+C`。

### 2. 夹爪合不上

- 必须 **长按左键**，短按会被下一帧 open 指令拉开
- 确认 Terminal 2 夹爪服务无 `Connection timeout`
- 确认用的是 Franka Hand，不是 Robotiq USB

### 3. 末端自己转

- 默认 `OSC_POSE` 含旋转通道，SpaceMouse 轴噪声会被放大
- 改用：`bash scripts/teleop_test.sh OSC_POSITION`

### 4. 参考位姿（桌面记录）

初始拍照位姿（桌面 `deoxys.txt`）：

```
0.05904578  0.52446592  -0.15976057  -0.76166065  0.18787272  1.7980539  0.81746185
```
